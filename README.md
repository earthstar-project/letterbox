# Letterbox

An app for thoughtful discussion for human-sized groups, powered by Earthstar.

- Post many threads per namespace
- Linear threads, no subthreads
- Supports Markdown
- Threads can be tagged with user-defined tags
- Per-post unread statuses

Earthstar enables the following:

- Livestreaming of changes
- Offline support
  - And stop all ingoing/outgoing messages even when online.
